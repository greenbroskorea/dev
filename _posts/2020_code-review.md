### 자주쓰는 IntliJ 단축키
- 라인 복제하기
  - MacOS: Cmd + d
  - Win/Linux: Ctrl + d
- 라인 삭제하기
  - MacOS: Cmd + 백스페이스
  - Win/Linux: Ctrl + y
- 이름 일괄 변경하기 (Rename)
  - MacOS: Shift + F6
  - Win/Linux: Shift + F6
- 라인별 코드 일괄 수정
   - Win/Linux: Alt + 마우스 드래그
- 전체 파일 목록 검색
   - Shift, Shift
